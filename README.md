# @flagkit

## Installation

**The react-native package requires `react-native-svg` v9 or higher**

### yarn

```sh
// react
yarn add @flagkit/react
// react-native
yarn add react-native-svg @flagkit/react-native
```

### npm

```sh
// react
npm i @flagkit/react
// react-native
npm i react-native-svg @flagkit/react-native
```
